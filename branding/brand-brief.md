# STEAMER Brand Brief

Brand personality:
- focused
- cinematic
- technical
- dependable

Visual direction:
- dark PC-launcher inspired surfaces
- steel-blue gradients with icy cyan highlights
- bold circular geometry and orbital motion
- trophy-led achievement cue embedded into the core mark
- achievement and progression cues without copying any platform mark

Icon concept:
- rounded square launcher tile
- large connected node system inspired by launcher / hardware iconography
- integrated trophy silhouette inside the primary node
- bright cyan focal points for small-size readability

Tone:
- premium desktop-gaming companion
- progress tracking and completion
- inspired by storefront launcher aesthetics, but original and copyright-safe
