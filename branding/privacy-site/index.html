STEAMER Privacy Policy

Effective date: 4/9/2026

Support contact: noahfoley6@gmail.com

STEAMER is an independent companion app designed to display public Steam profile, library, and achievement data in a mobile-friendly dashboard.

What data the app uses

  • Steam ID
  • Public Steam profile data
  • Owned games list
  • Achievement data for supported games

How data is used

  • To load your Steam profile
  • To show your library and achievement progress
  • To cache data locally on your device for faster loading

What the app does not store

  • Steam passwords
  • Payment information
  • Private messages

Local storage

STEAMER stores cached Steam profile, game, and achievement data locally on your device using app preferences. Signing out clears the locally stored session and cached app data.

Backend services

STEAMER uses a Cloudflare Worker to proxy Steam Web API requests that require a server-side API key.

Third-party services

  • Cloudflare, for backend request handling
  • Valve/Steam, for public profile and achievement data

Changes

This privacy policy may be updated from time to time.
